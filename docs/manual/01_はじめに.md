# はじめに

## PKC2 とは

**PKC2（Portable Knowledge Container 2）** は、個人の知識・メモ・タスク・添付ファイルを 1 つの HTML ファイルにまとめて管理するためのツールです。

- **単一 HTML** — アプリケーション本体とデータの両方が 1 つの `.html` ファイルに収まります
- **ローカル完結** — サーバーもインターネット接続も不要。ブラウザさえあれば動きます
- **オフライン対応** — USB メモリ、メール添付、社内共有ドライブなど、HTML ファイルを受け渡せる環境ならどこでも使えます
- **自動保存** — IndexedDB に変更が即座に保存されます
- **持ち出し可能** — HTML ファイルや ZIP パッケージとして書き出して配布・バックアップできます

## 設計思想

PKC2 は、次の 5 つの原則に基づいて作られています（詳細は開発者向け文書 `docs/planning/05_設計原則.md` を参照）。

1. **単一 HTML 自己完結** — 外部依存ゼロ。1 つの HTML ファイルが完全なアプリです
2. **非破壊のリビジョン** — 編集前の状態がリビジョン履歴として残り、後から復元できます
3. **データ一貫性** — Container という 1 つの集約がデータの正本。UI 状態と永続データは明確に分離されています
4. **可搬性** — データはいつでも HTML / ZIP として持ち出せ、別の環境に移せます
5. **段階的進化** — 破壊的変更を避け、古いデータが新しい PKC2 でも読めることを重視します

## PKC2 でできること

- **メモ・ノート** を Markdown で書く（text archetype）
- **日誌・ログ** をタイムスタンプ付きで追記する（textlog archetype）
- **ToDo** を期日・ステータス付きで管理する（todo archetype）
- **構造化データ** を name / note / checked の簡易フォームで記録する（form archetype）
- **ファイル添付** として画像・PDF・音声・動画などを保存する（attachment archetype）
- **フォルダ** でエントリを階層的に整理する（folder archetype）
- **ビューの切替** — 詳細ビュー・カレンダー・カンバンで同じデータを違う切り口で見る
- **検索・フィルタ・ソート** — タイトル・本文の全文検索、archetype 絞り込み、タグ絞り込み
- **関連付け** — エントリ間を構造的・カテゴリ的・意味的・時間的に結びつける
- **リビジョン履歴** — 編集前の内容を自動で保存、必要なときに復元
- **Export / Import** — HTML（軽量版・完全版）や ZIP パッケージとして書き出し、別環境で読み込み
- **Rehydrate** — 閲覧専用 HTML を編集可能なワークスペースに昇格
- **埋め込み** — iframe で別ページに PKC2 を埋め込み、postMessage で親ページと通信

## PKC2 でできないこと（意図的な制約）

- **クラウド同期** — データはブラウザの IndexedDB に保存されるため、クラウド上での自動同期はありません。複数デバイス間での同期は Export / Import で手動で行います
- **マルチユーザー同時編集** — 共同編集機能はありません。1 ファイル = 1 ワークスペースの個人ツールです
- **サーバー連携** — サーバーを介した認証・権限管理・中央集約はありません
- **モバイルアプリ** — ブラウザで動作しますが、ネイティブアプリはありません
- **動的スキーマ拡張** — form archetype は固定フィールド（name / note / checked）のみ。自由なスキーマ設計はできません

これらは将来の拡張として除外しているわけではなく、「単一 HTML 自己完結」の原則を守るための意図的な選択です。

## システム要件

- **ブラウザ**: Chrome 80+、Firefox 113+、Safari 16.4+、Edge 80+ を推奨
  - IndexedDB 対応が必須
  - Export の完全版で `CompressionStream` API を使うため、上記バージョン以上が推奨されます
- **ディスク**: ファイル添付を多用する場合はブラウザの IndexedDB 容量（通常数百 MB〜数 GB）に依存
- **インターネット接続**: 不要（初回ダウンロード時のみ）
