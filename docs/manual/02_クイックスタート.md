# クイックスタート

3 分で PKC2 を動かすための最短手順です。

## 1. PKC2 を入手する

PKC2 のリポジトリから `dist/pkc2.html` をダウンロードします。

- GitHub 上で `dist/pkc2.html` を表示 → 右上の **Download raw file** ボタンを押す
- または、リポジトリを git clone して `dist/pkc2.html` を直接取り出す

PKC2 のインストール作業は不要です。単一 HTML ファイル 1 つで完結します。

## 2. ブラウザで開く

ダウンロードした `pkc2.html` をダブルクリックすると、既定のブラウザで開きます。

- ブラウザのアドレスバーには `file:///.../pkc2.html` のように表示されます
- 初回起動時は空のワークスペースが表示されます
- サイドバー（左側）にエントリ一覧、詳細ペイン（右側）に選択中のエントリが表示されます

## 3. 最初のエントリを作成する

### テキストノートを作る

1. 画面上部の **+ Note** ボタンを押す
2. タイトル入力欄にタイトル（例: 「初めてのノート」）を入力
3. 本文欄に内容を記入（Markdown が使えます）
4. **保存** ボタン、または `Ctrl+S` / `Cmd+S` で確定

### Todo を作る

1. 画面上部の **+ Todo** ボタンを押す
2. タイトル欄に Todo の内容（例: 「買い物に行く」）を入力
3. 必要なら期日を設定
4. **保存** で確定

## 4. 自動保存を確認する

PKC2 は変更のたびに自動で保存します（約 300 ミリ秒の遅延あり）。

1. 何かエントリを作成・編集する
2. ブラウザのタブを閉じる
3. `pkc2.html` を再度ブラウザで開く
4. 先ほど作ったエントリが残っていれば、自動保存が正しく働いています

> データは **ブラウザの IndexedDB** に保存されます。同じブラウザ・同じ HTML ファイルの組み合わせでないと復元されない点に注意してください。持ち出すには Export を使います（[07 保存と持ち出し](07_保存と持ち出し.md) 参照）。

## 5. ビューを切り替える

画面上部のビュー切替ボタンで、データを違う切り口から見ることができます。

- **Detail** — エントリ一覧と選択中のエントリの詳細
- **Calendar** — 期日付きの Todo をカレンダー表示
- **Kanban** — Todo を「Todo」「Done」の 2 列に並べるカンバン

どのビューで操作しても、データは 1 つのワークスペースに統合されます。
